# Mern-AI

And AI agent created using MERN stack and openAI.

Installation:

- npm i on both server and client.
- Download ollama for windows -> run the installer -> run following command on powershell "C:\Users\<system-name>\AppData\Local\Programs\Ollama\ollama.exe" pull llama3
  -run "npm run dev" on client and "node server.js" on server.
